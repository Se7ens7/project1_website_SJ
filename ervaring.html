Ervaring

Tot op heden heb ik enkele lessen in verschillende programmeertalen (HTML, CSS, C# en SQL) achter de rug, ook kan ik vlot blindelings typen. Ik deed wel ervaring op in de horeca, hier heb ik 6 stageperiodes doorlopen alsook heb ik vakantiewerk gedaan. Ik heb deze opleiding 7 jaar lang gevolgd op de hotelschool in turnhout waar ik ook mijn secundair diploma heb behaald.
